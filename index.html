COMP2011 Lab 9

Introduction

This lab will help you to practice C++ linked list.

Problem Description

Assume there are 7 students in the linked list. Struct{name, year, ptr1, ptr2}: Chase, 12, Dahua, 15 Fred, 16 Yang, 17 Liz, 14 Daisy, 13 Sam, 15 David, 13

Tasks

We have implemented a code skeleton for this problem (lab9_skeleton.zip), in which some parts are left blank with necessary comments. You need to fill in the blanks marked with TODO.

You are not allowed to move any function definition to other files (but you can declare functions in different files).

Specifically, there are five TODOs:

  1. Implement Add() function in xxx.hpp: given the structure, you can add a new student with information Struct{name, year, ptr1, ptr2}.
  2. Implement Delete() function in xxx.hpp: you can delete a student with all infomation.
  3. Implement Print() function in xxx.hpp: it can print all the students in some year.
  4. Implement Search() function in xxx.hpp: check the student in the linked list, then return the students informaton
  5. You are supposed to check memory leaks in both main.cpp and xxx.hpp, i.e. free all the memory blocks that are dynamically allocated.
